# Product

Mosaic (identifier: `com.zimei.mosaic`) is a desktop application built with Tauri v2 and React. The project is in early development — currently scaffolded from the Tauri + React + TypeScript template with minimal custom functionality.

The `docs/` folder contains reference documentation for OpenAI's Codex CLI project (a Rust-based AI programming assistant). These docs serve as study/reference material and do not describe this application's architecture.

## Current State

- Starter template with a single `greet` Tauri command
- No routing, state management wiring, or feature modules implemented yet
- Zustand and Immer are installed as dependencies, suggesting planned state management
- lucide-react is installed for iconography
